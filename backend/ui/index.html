LLM ENGINE
配置系统内部需要大语言模型的引擎 — 分类器 (Classifier) 和 知识整合 (Reflection)
🤖
为什么需要配置 LLM 引擎？
分类器（classifier.py）在写入记忆时自动将内容分类为「常识/人物/代码」等类型，需要一个轻量 LLM。
知识整合引擎（reflection）定期分析全量记忆、合并重复、发现关联，需要一个推理能力强的 LLM。
两者均属于系统内部调用，费用由系统所有者承担（或配置自托管模型免费使用）。
🏷️ 分类器 LLM (Classifier Engine) classifier.py
推荐: DeepSeek-chat / GPT-4o-mini（轻量快速，成本极低）
🔮 知识整合 LLM (Reflection Engine) reflection
推荐: DeepSeek-reasoner / GPT-4o（需要较强推理能力）
