外网/跨地域访问指南

当前系统已监听 0.0.0.0:8003。若需从公网访问：

  • 端口转发: 在路由器中将 8003 端口转发至本机。
  • 内网穿透: 推荐使用 ngrok 或 cpolar 将本地服务映射至公网域名。
  • 虚拟局域网: 使用 Tailscale 或 ZeroTier 建立加密通道，实现跨地域零配置访问。
